# b4geoip

b4geoip является расширенной версией geoip.dat библиотеки от проекта [RUNETFREEDOM](https://github.com/runetfreedom/russia-blocked-geoip).

Дополнительно включает в себя категории:

- geoip:github
- geoip:oracle
- geoip:roblox
- geoip:amazon

Прямая ссылка на последнюю версию (обновляется каждые 3 часа):

```text
https://github.com/DanielLavrushin/b4geoip/releases/latest/download/geoip.dat
```
